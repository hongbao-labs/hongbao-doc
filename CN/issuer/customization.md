# 卡面联名定制

Hongbao 卡面大面积留白，绝大部分区域开放给项目方做品牌化设计。只有极少数必要元素由 Hongbao 统一保留，用于确保持卡人能认出这是一张可信红包卡，并能正常完成领取。

> 这是一份正在完善中的规格文件。标注 **🚧 即将公布** 的规格、报价与流程尚未最终对外公布，以最新商务确认稿为准。需具体报价或设计模板，请走 [contact.md](../contact.md)。

---

## 保留区域（不可变）

| 位置 | 内容 | 原因 |
|---|---|---|
| **卡片正面 左上角** | Hongbao logo | 持卡人识别可信红包卡的锚点 |
| **卡片背面** | 二维码（指向链上地址 / 领取入口） | 持卡人扫码领取的唯一入口 |
| **卡片背面** | 必要文字信息（操作提示 / 安全提醒 / 其他，🚧 即将公布）| 合规与安全提示 |

> 🚧 即将公布：卡片正反面示意图，标注保留区域的精确尺寸与位置。

## 可定制区域（开放给项目方）

| 区域 | 可定制项 |
|---|---|
| **正面**（除左上角 logo 外的全部区域） | 主视觉、品牌 logo、活动标语、配色、特殊工艺 |
| **背面**（除二维码 + 保留文字外的留白） | 项目方说明文字、社媒账号、活动信息（字数 / 字号上限 🚧 即将公布） |

不可定制项（由硬件 / 协议层决定）：

- 卡片尺寸 / 厚度：🚧 即将公布
- 二维码 / 按键 / 指示灯的位置和尺寸
- 芯片型号、固件、蓝牙协议

## 流程与时间线

```
1. 商务对接 → NDA / 合同节奏（即将公布）
2. 设计稿提交 → 项目方按 Hongbao 模板提交（格式见下方「设计稿规范」）

5. 量产 → 量产工期：即将公布
6. 出货 → DOA 政策按合同
```

## 设计稿规范

> 以下文件格式、出血、分辨率、色彩模式为印刷通用标准，供项目方先行准备稿件；最终以 Hongbao 模板与打样校色为准。

| 项 | 要求 |
|---|---|
| **文件格式** | AI / PDF / EPS 等矢量格式；所有文字转曲，链接图片随稿打包 |
| **出血** | 四周各预留 3 mm 出血，重要内容距裁切线 ≥ 3 mm |
| **分辨率** | 嵌入的位图元素按成品实际尺寸 ≥ 300 dpi |
| **色彩模式** | CMYK；如需品牌专色，另标注 Pantone 色号 |
| **必要保留区** | 左上角保留区（尺寸 🚧 即将公布）内不能放任何元素；背面保留区（区域描述 🚧 即将公布）内不能压字 |

## 常见问题

**Q：可以完全去掉 Hongbao logo 吗？**
不可以。左上角 logo 是持卡人识别可信红包卡的唯一锚点，所有出货卡片必须保留，不接受商务免除。

**Q：打样要多久？打样费可以抵扣量产订单吗？**
打样工期与打样费是否抵扣量产单按订单确认，请联系 bd@hongbao.digital（详见 [contact.md](../contact.md)）。

**Q：能给同一批卡内做多种设计（变体）吗？**
是否支持、多设计的起订量与额外费用按订单评估，请联系 bd@hongbao.digital（详见 [contact.md](../contact.md)）。
